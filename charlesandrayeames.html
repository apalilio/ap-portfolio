Charles and Ray Eames Editorial

The goal of this project was to create multiple layouts using the same content, while keeping in mind all the rules of typography I had learned thus far. This was my first typography project that combined type with imagery, which made it both exciting and challenging. I found myself navigating the many possibilities for layout while ensuring that the integrity of the text remained clear and undistorted. It pushed me to think more critically about how type and visuals interact within a design.
